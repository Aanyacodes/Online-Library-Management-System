# Online Library Management System

## Overview
This project is a Java-based Online Library Management System developed as a BCA Final Year Project.

The system helps manage:
- books
- users
- librarians
- transactions
- borrowing and returning operations

---

## Features
- User Login
- Librarian Dashboard
- Book Management
- Transaction Handling
- Member Portal
- Database Connectivity

---

## Technologies Used
- Java
- Maven
- HTML
- MySQL

---

## Project Structure
- DAO Architecture
- Maven-based project structure
- Separate UI pages

---

Project synopsis and report are available in the `documentation` folder.
